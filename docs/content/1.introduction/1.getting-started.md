# Getting Started
The library is available on [npm](https://www.npmjs.com/package/very-good-fetch) and can be installed with: 


::code-group

  ```bash [npm]
  npm install very-good-fetch
  ```

  ```bash [yarn]
  yarn add very-good-fetch
  ```

  ```bash [pnpm]
  pnpm add very-good-fetch
  ```

::
