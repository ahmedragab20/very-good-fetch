# Getting Started
The library is available on [npm](https://www.npmjs.com/package/@nuxtjs/docus) and can be installed with: 

1. Install the library:

::code-group

  ```bash [npm]
  npm install very-good-fetch
  ```

  ```bash [yarn]
  yarn add very-good-fetch
  ```

  ```bash [pnpm]
  pnpm add very-good-fetch
  ```
